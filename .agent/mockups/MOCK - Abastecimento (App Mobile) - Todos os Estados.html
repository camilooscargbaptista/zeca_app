Condições Dinâmicas da Tela
  • Estado 1: Se _vehicleData == null → Mostra alerta "Nenhuma jornada ativa" + botão "Iniciar"
  • Estado 2: Se _vehicleConfirmed && !_isStationValidated → Mostra input de CNPJ
  • Estado 3: Se erro ao validar CNPJ → Mostra error-box com mensagem
  • Estado 4: Se _selectedFuel.isEmpty (sem compatível) → Mostra error-box
  • Estado 5: Se _vehicleConfirmed && _isStationValidated && _selectedFuel.isNotEmpty → Tudo OK, botão ativo
  • Estado 6: Se _showKmWarning → Mostra warning-box laranja
  • Último KM: Aparece no card Veículo (sempre) + badge no form (se _lastOdometer != null)
  • ARLA: Checkbox só aparece se _hasArlaAvailable && combustível.contains('diesel')
  • Dropdown Combustível: Se _availableFuels.length > 1 → Dropdown; senão → readonly
  • Badge Posto: Parceria (verde), Autônomo (laranja), Frota (azul)
  • Botão: Sempre fixo. Cor #2A70C0 (azul ZECA). Desabilitado quando condições não atendidas.
